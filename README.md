
# Movie Recommonder Sytem

A content-based movie recommendation engine that provides personalized film suggestions based on user input. The system leverages Natural Language Processing (TF-IDF Vectorization) and K-Nearest Neighbors to analyze a dataset of over 45,000 movies. It integrates the TMDb API for real-time metadata retrieval, offering a seamless user experience through a responsive Streamlit web interface.




## Features

- Content-based filtering using TF-IDF and K-Nearest Neighbors.
- Real Time API Integration
- Interactive Web Interface
- Scalable data handling


## Tech Stack

**Frontend:** Streamlit

**Backend:** Python

**Machine Learning:** Scikit-Learn (TF-IDF, Nearest Neighbors)

**API Integration:** TMDb REST API


## Run the Project

Clone the project

```bash
  git clone https://github.com/hxrdikk21/movies-recommendation-system/
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  streamlit run app.py
```
